# Desafios de MLOps — Do Treino à Produção

Este repositório contém os desafios práticos focados em transformar modelos de Machine Learning em produtos de software reais, utilizando práticas modernas de MLOps.

---

## 1. O Identificador de Pinguins (FastAPI + Pydantic)
- **Dataset:** Palmer Penguins.
- **Foco:** Validação de Dados (Pydantic).

---

## 2. Eficiência Energética (Model Artifacts & Joblib)
- **Dataset:** Energy Efficiency (UCI).
- **Foco:** Serialização e Artefatos (Joblib).

---

## 3. Previsão de Preço de Diamantes (Docker Básico)
- **Dataset:** Diamonds.
- **Foco:** Conteinerização (Docker).
- **Nota:** O uso de Docker é obrigatório para execução.

---

## 4. Diagnóstico de Saúde do Coração (Testes Unitários)
- **Dataset:** Heart Disease (UCI).
- **Foco:** Qualidade de Software e Validação Biológica (Pytest).

---

## 5. Avaliação de Compra de Carros (Linting)
- **Dataset:** Car Evaluation (UCI).
- **Foco:** Qualidade de Código e Padronização.
- **Ferramentas:** Flake8 + GitHub Actions.

---

## 7. Detector de Spam e Esteira Automática (CI/CD)
- **Dataset:** SMS Spam Collection (UCI).
- **Foco:** Automação Completa (CI/CD).
- **Status:** Projeto robusto pré-existente.
